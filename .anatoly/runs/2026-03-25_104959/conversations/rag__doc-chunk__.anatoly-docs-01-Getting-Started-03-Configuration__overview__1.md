RustGuard supports two mutually exclusive operating modes, each with its own configuration surface:

- **Standard tunnel mode** — reads a wg.conf-format file, identical to upstream WireGuard. Managed by the rustguard-daemon crate.
- **Zero-config enrollment mode** — no config files; keys and IPs are negotiated at runtime over an encrypted enrollment protocol. Managed by the rustguard-enroll crate.

Both modes are accessed through the same rustguard-cli binary. The mode is selected by the subcommand used.

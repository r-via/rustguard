After the server is running, enrollment is managed through a UNIX domain control socket using the following subcommands.

### Open the enrollment window

Opens the enrollment window for the specified number of seconds. The window closes automatically when the timer expires. Existing connected peers are not affected.

### Close the enrollment window immediately

Closes the enrollment window before it expires naturally. Peers that completed enrollment before this call remain connected.

### Inspect current state

Prints the current enrollment window state (open/closed, seconds remaining) and the count of enrolled peers.

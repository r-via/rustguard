### rustguard-core

Implements the Noise\_IKpsk2 handshake, transport sessions, replay window, and timer state machine. Dual std/no_std.

#### Replay Window

The replay window is a 2048-bit sliding bitmap. Verification and state mutation are separated to prevent window poisoning by unauthenticated packets.

#### Handshake State

Handshake chaining key, hash, and PSK are ZeroizeOnDrop — secret material is cleared from memory when the struct is dropped.

#### Cookie Mechanism (DoS Protection)

MAC1 is verified before any DH operations are performed. MAC2 is required when the server is under load.
